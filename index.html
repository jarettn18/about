Tiny Compiler

Python 3, C

This project showcases the inner workings of a compiler and the entire process of compiling code. Here we designed our own NP-Complete programming language and wrote our own compiler to parse the code, generate a syntax tree, and walk the tree to generate ASM instructions. We applied our fundamental understandings of programming languages and compilers to understanding how we can go from theory to code generation.

Image Rasterizer

C++, OpenGL

In this project, we wrote a program to render images using the scanline algorithm. The scanline algorithm was one of the first techniques used for image rendering where it takes an input of triangle coordinates and colors the pixels and draws lines to render images line by line on a screen. We also used OpenGL to showcase how light can reflect off an object in virtual space.
